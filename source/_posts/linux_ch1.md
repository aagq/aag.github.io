# chapter1 What is Linux?

## Linux 是什么
    
  1.  是一组软件
  2.  操作提同除了有效率地控制这些硬件资源的分配，并且提供计算机运作所需要的功能（除网络功能）。
  3.  Linux 是 open source 的操作系统，所以它的程序可以被修改成适合在各种机器上面运行，即 具有 **可移植性** 。

## Linux 的核心版本

	 2.6.18-92.el5
	主版本.次版本.释出版本-修改版本
    	- 主、次版本为奇数：发展中的版本
	- 主、次版本为偶数：稳定版本

注：通常所说的Linux 版本指的是核心版本。还有，Linux 其实就是一个操作系统最底层的核心及其提供的核心工具。
